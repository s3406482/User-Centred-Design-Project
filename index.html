Main Menu

This is the main page or the main menu screen. The menu items will be listed and selecting the item will take you to the item page. The help button will either call a staff member to manually take the order or it can display a help page

Item page

This is the item page it will display a picture of the item on the right and on the left will list the options ie: which side, how you want it cooked, what sauce to have etc. You can add it to the order here or you can select to add/remove selected ingredients

Add/Remove items

This is the page where we can add/remove ingredients. The already selected ingredients are those that are on the item currently. And the unselected ingredients are those you can add. The forward/back arrows allow you to scroll through multiple ingredients. They will disappear if not needed

Cart

This is the cart page it will display all the current order contents. Display the automatically generated order no. and will allow you to select the payment options. Selecting continue order will take you back to the main screen

Final Page

This is the final page it will display an order summary and allow you to order again if you would like too. This will take you back to the main screen. The image will be a Thanks for ordering with a picture of the restaurants logo
